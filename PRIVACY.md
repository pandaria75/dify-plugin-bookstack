# Privacy Policy

## Data collection

This plugin does **not** collect, store, log, or transmit any personal data from users. It does not gather direct identifiers (names, email addresses, phone numbers), indirect identifiers (IP addresses, device IDs, location data), or combinable data that could identify an individual.

## How the plugin works

This plugin connects only to the BookStack instance configured by the user.

The plugin may read, create, and update BookStack pages, books, and chapters according to the tools the user invokes.

The plugin uses API credentials supplied by the user through Dify plugin credential settings.

## Third-party services

The plugin sends requests to the user's own BookStack API instance. No data is sent to any other third party.

The plugin runs inside the Dify runtime, which executes the plugin. How Dify itself handles data is covered by the Dify platform's own privacy policy.

## Credential handling

The plugin does not log `token_secret` values.

## How content is used

How page content is later used by Dify Workflow, Chatflow, Agent, or Knowledge features depends on the user's own Dify configuration and data handling choices.
