# Privacy Policy

This plugin connects only to the BookStack instance configured by the user.

The plugin may read, create, and update BookStack pages, books, and chapters according to the tools the user invokes.

The plugin uses API credentials supplied by the user through Dify plugin credential settings.

The plugin does not intentionally send content to any third party other than the configured BookStack instance and the Dify runtime that executes the plugin.

The plugin does not log `token_secret` values.

How page content is later used by Dify Workflow, Chatflow, Agent, or Knowledge features depends on the user’s own Dify configuration and data handling choices.
