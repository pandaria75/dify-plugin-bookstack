# Tools

## Implemented Tool Plugin Features

### Core page operations

- `validate_credentials`
- `search_pages`
- `get_page`
- `create_page`
- `update_page`
- `publish_page`

### Support listing tools

- `list_books`
- `list_chapters`
- `list_shelves`
- `list_pages`

## Usage Notes

- `publish_page` supports create-or-update behavior.
- `list_books`, `list_chapters`, `list_shelves`, and `list_pages` are implemented support tools.
- Delete or archive operations are not part of the current plugin scope.
