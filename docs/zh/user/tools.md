# 工具

## 已实现的 Tool 插件功能

### 核心页面操作

- `validate_credentials`
- `search_pages`
- `get_page`
- `create_page`
- `update_page`
- `publish_page`

### 支持性列表工具

- `list_books`
- `list_chapters`
- `list_shelves`
- `list_pages`

## 使用说明

- `publish_page` 支持 create-or-update 行为。
- `list_books`、`list_chapters`、`list_shelves` 和 `list_pages` 是已实现的支持工具。
- 删除或归档操作不属于当前插件范围。
