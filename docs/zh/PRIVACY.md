# 隐私政策

该插件仅连接到用户配置的 BookStack 实例。

根据用户调用的工具，该插件可能读取、创建和更新 BookStack 的页面、图书和章节。

该插件使用用户通过 Dify 插件凭据设置提供的 API 凭据。

除已配置的 BookStack 实例以及执行该插件的 Dify 运行时之外，插件不会有意将内容发送到任何第三方。

该插件不会记录 `token_secret` 值。

本仓库还包含关于打包、Marketplace 就绪性以及 Datasource 后续工作的独立文档和规划材料，但这些文档不会改变此处描述的当前隐私立场。

页面内容之后如何被 Dify Workflow、Chatflow、Agent 或 Knowledge 功能使用，取决于用户自己的 Dify 配置与数据处理选择。
