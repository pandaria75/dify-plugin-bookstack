# 018. Add GitHub issue templates

- **Priority:** P1
- **Type:** Repository Maintenance

## Background

Issue templates help contributors report bugs and request features consistently.

## Goal

Add GitHub issue templates for bugs, features, and questions.

## Implementation Suggestion

- Add simple issue form templates.

## Acceptance Criteria

- Contributors can file issues with consistent structure.

## Risk

Templates should not become too complex for a small project.
