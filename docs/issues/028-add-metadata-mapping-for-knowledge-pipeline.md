# 028. Add metadata mapping for Knowledge Pipeline

- **Priority:** P2
- **Type:** Data Mapping

## Background

RAG ingestion requires stable metadata for filtering and traceability.

## Goal

Document and implement a metadata mapping scheme for BookStack content.

## Implementation Suggestion

- Include source, IDs, title, URL, tags, and timestamps.

## Acceptance Criteria

- Metadata output is stable and documented.

## Risk

Some BookStack fields may not always be available for every record.
